# LITTLE RED ROBOT 3D
[Download game in Rar file](https://github.com/DamianPyCoder/Unity__Games__x6/tree/main/red_robot_executable)  
The code is in a privat repository.

### How to play:
```
   Move left:        A
   Move right:       D
   Move down:        S
   Move up:          W
   To roll:          Space
   Attack:           Mouse click
```
